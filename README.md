# Tax Empire: Panic Season

A fast, arcade-style Node.js simulation game where you run a chaotic tax office.

## What's new in this redesign

- Interactive **office stations** instead of a text-heavy dashboard
- Pop-up actions for **Client Inbox, Prep Desk, IRS Mail, Staff Board, and Upgrades**
- Visual HUD with live meters for cash, stress, reputation, and audit risk
- **Audit mini-game**: quickly click the right document under a timer
- Mode system: **Career**, **Hardcore**, **Sandbox**
- Persistent save via `localStorage`

## Run

```bash
npm start
```

Open `http://localhost:3000`.

## Check syntax

```bash
npm run check
```
